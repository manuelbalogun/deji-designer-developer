WORKFLOW

My Process. 🙇🏽

01

Research & Discovery

This is the strategic/foundational phase where I make sure to get all the facts and data right. Idea Validation, Competitive Analysis, and User/Product Research.

02

Ideation & User Experience

After understanding the 'problem' and how to go about solving it, I work with the data I have to create User Stories, User Journey Maps/User Flow Diagrams and Wireframes.

03

Interface & Interaction

With the research and user experience solidified, I proceed to create the user interface and accompanying interactions, with adequate testing where applicable.
